刻意練習（下）

接著我們將講到本書精髓 - 刻意練習
如果還沒看過刻意練習（上）很建議先去看那篇文章。

刻意練習能改變生理狀態​

在這個段落，作者舉了大量實驗和故事證明人體的適應力極強。舉例來說，許多科學家、音樂家、醫生的大腦的某個關鍵區域都比常人還更活躍或是更大，這就是大腦隨著不斷練習漸漸進化的證明。

許多看似不可能的能力（例如記住紐約市所有街道名稱），其實不是做不到，而是「目前」你無法做到。不過你絕對可以透過刻意練習來增強相關的能力，甚至促使生理改變（科學家的大腦、運動員的肌肉…）。

經驗不會讓你越做越好​

這是一個顛覆傳統思維的想法 - 經驗不會讓你越做越好。

作者舉了許多例子或實驗證明這個觀點。例如...

  • 開車二十年的駕駛未必比開車兩年的駕駛技術更佳
  • 執業二十年的醫生的診斷能力甚至比執業兩年的醫生差

書中還有無數個例子，建議大家自己去看會更有說服力。但結論為 - 單純埋頭猛幹是不會進步的，你得有意識的在每一次執行中尋找可以進步的地方並改善，這樣才會進步，這也就是所謂的刻意練習。例如你想要鋼琴彈得好，每天瘋狂彈是不會進步的（或是進步極慢），你得找出總是彈不好的那個小節花心思練習才行。練習的意識和品質才是重點。

心智表徵​

這個概念很抽象且難以描述，但我還是盡量說明我的理解。

心智表徵的品質即為常人與專家的主要差異。

所謂「心智表徵」大概就是在執行一件事時，清楚知道前一步是什麼、現在正在做什麼、以及下一步可能發生什麼。這並不是單純死記硬背，而是以模組的概念記憶資訊。例如職業棋手不會分開記憶每個旗子的位置，而是將某一區域的旗子「當下的狀態記下來」，這些旗子都是相互關聯並且具有意義的。

心智表徵佳的人處理資訊是系統化的，所有資料相互有意義而且可以來往，因此他們可以把訊息登錄進長期記憶。同時專家可以分類不同的資訊並給予出不同的處理方式。當系統已經烙印在腦海中，專家的思考過程將自動化，所以有時他們也無法解釋為什麼做出當下的判斷，只能解釋「就是有一種直覺」，但這些直覺往往非常精準。

生活中的心智表徵呈現

有沒有曾在生活中遇到「看起來超輕鬆地完成超難任務」的人？
我認為這就是「高品質的心智表徵」顯現在生活中的樣子，分辨真正的專家這篇文章就在討論這個主題，有興趣可以去看看。

如何進行刻意練習？​

接下來是本書重點「怎麼刻意練習」

這段會告訴你超越常人的「天才們」到底是怎麼做到的，其實並不是他們受上天眷顧或是他們有魔法之類的，只是因為他們懂訓練的方法，這是很科學的。

大概有幾個重點...

  • 絕對的專注：作者認為專注度不夠的練習是沒有效用的。每一次的練習，你都必須全心全意付出在當下，專心思考現在正在執行的動作。

  • 每次練習都要超出原本能力一點點：剛好超出舒適圈一點點就是練習的最佳狀態（超出能力太多的訓練也會導致練習失效），同時也是最容易達到「心流」的狀況。書裡說「當你開始分心或是覺得輕鬆好玩時，大概就代表沒有做對」，這聽起來很累人也不討喜，但這就是養成能力的唯一方法。如果你想透過練習進步，那練習絕對不會是一件輕鬆愉快的事。

  • 回饋機制：假如你想練習籃球罰球，你不可能投球之後不看自己有沒有投進吧？ 這個概念簡單易懂，不過只要換一個領域人們就常常忘記這個觀念。例如猛彈鋼琴不注意哪裡彈得不對、猛寫文章可是不知道哪裡寫得好哪裡不好、放射師對 X 光片進行預測卻不知道病人後續的狀況（所以他們根本不知道自己有沒有判斷對）……

    當你沒有回饋機制，你將無法改善自己的錯誤，練習將徒勞無功。相反的，你應該利用爾偶跳出舒適圈遠一點，找出最容易出錯的部分，並有意識地加強這個部分。在改善時，「念出來」這個方法極為有用，因為念出來可以強迫你的大腦注意到當下執行的動作。例如每次鋼琴彈到一個地方手都忘記抬起來，你就規定自己彈到這邊就要大聲唸出「手抬高」，就是一個很棒的刻意練習。

總結刻意練習，就是清楚知道「要練什麼」然後練它，
還有知道「什麼是弱點」然後練它。
